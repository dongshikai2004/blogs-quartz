# 语音信号的格式
WAV，PCM, MP3，m4a, AIFF, WMA, Ogg
WAV 去掉文件头，只保留数据部分->PCM
WAV 16bit均匀量化编码，每两个字节表示一个采样点
采样点取值范围-$32767$~$32768$,浮点表示$-1$~$+1$
# 语音信号的读取
`librosa`只支持wav
## 文件读取
```python
librosa.load(path,*,sr=22050,mono=True,offset=0.0,duration=None,dtype=<class 'numpy.float32'>,res_type  
='soxr_hq')
```
### 参数
- **path**：字符串或文件对象，表示音频文件的路径或文件对象。
- **sr**（采样率）：整数，默认值为 22050。如果指定为 `None`，则使用音频文件的原始采样率。如果指定为一个整数值，则将音频文件重新采样到这个采样率。
- **mono**（单声道）：布尔值，默认值为 `True`。如果为 `True`，则将音频文件转换为单声道；如果为 `False`，则保留音频文件的原始声道数。
- **offset**：浮点数，默认值为 0.0。表示从音频文件的哪个时间点开始加载，单位为秒。
- **duration**：浮点数，默认值为 `None`。表示加载音频文件的持续时间，单位为秒。如果为 `None`，则加载整个音频文件。
- **dtype**：数据类型，默认值为 `np.float32`。指定返回的音频数据的类型。
- **res_type**：字符串，默认值为 `'kaiser_best'`。指定重采样方法。常用的选项有：
    - `'kaiser_best'`：高质量重采样。
    - `'kaiser_fast'`：快速重采样。
    - `'scipy'`：使用 SciPy 的重采样方法。
    - `'polyphase'`：多相滤波器组重采样。
    - `'linear'`：线性插值重采样。
    - `'zero_order_hold'`：零阶保持重采样。
### 返回值
- **y**：一维 NumPy 数组，表示音频信号的波形数据。如果 `mono=True`，则返回单声道音频数据；如果 `mono=False`，则返回多声道音频数据。
- **sr**：整数，表示音频的采样率。
## 波形显示
### matplot直接显示
### waveshow显示
```python
librosa.display.waveshow(y,*,sr=22050,max_points=11025,axis='time',offset=0.0,marker='',where='post',la  
bel=None,transpose=False,ax=None,x_axis=<DEPRECATED parameter>,**kwargs)
```
#### 参数
- **y**：一维 NumPy 数组，表示音频信号的波形数据。
- **sr**（采样率）：整数，表示音频的采样率。
- **max_points**：整数，默认值为 10000。表示在波形图中显示的最大点数。如果音频信号的长度超过这个值，`librosa.display.waveshow()` 会自动对音频信号进行下采样，以便在图中显示。
- **x_axis**：字符串，默认值为 `'time'`。指定 x 轴的标签。常用的选项有：
    - `'time'`：显示时间轴。
    - `'off'`：不显示 x 轴标签。
- **offset**：浮点数，默认值为 0.0。表示波形图的偏移量，单位为秒。
- **marker**：字符串，默认值为 `None`。指定波形图中数据点的标记样式。如果为 `None`，则不显示标记。
- **label**：字符串，默认值为 `None`。指定波形图的图例标签。
- **alpha**：浮点数，默认值为 1.0。指定波形图的透明度，范围为 0.0 到 1.0。
- **fill**：布尔值，默认值为 `True`。如果为 `True`，则填充波形图的上下部分。
- **linewidth**：浮点数，默认值为 1.0。指定波形图的线宽。
- **linestyle**：字符串，默认值为 `'-'`。指定波形图的线型。
- **color**：字符串，默认值为 `None`。指定波形图的颜色。如果为 `None`，则使用默认颜色。
- **ax**：Matplotlib 的 Axes 对象，默认值为 `None`。指定在哪个坐标轴上绘制波形图。如果为 `None`，则使用当前的坐标轴。
#### 输出
在当前的 Matplotlib 图形窗口中绘制波形图
