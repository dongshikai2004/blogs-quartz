# 静音消除（去除前后的静音）
```python
librosa.effects.trim(y, *, top_db=60, ref=<function amax>, frame_length=2048, hop_length=512,  
aggregate=<function amax>)
```
### 参数

- **y**：一维 NumPy 数组，表示音频信号的波形数据。
- **top_db**：浮点数，默认值为 60。表示静音部分的阈值，单位为分贝（dB）。低于这个阈值的音频部分被认为是静音。
- **frame_length**：整数，默认值为 2048。表示用于计算能量的帧长度。
- **hop_length**：整数，默认值为 512。表示帧之间的跳数。

### 输出

- **y_trimmed**：一维 NumPy 数组，表示去除静音部分后的音频信号。
- **index**：元组，表示去除静音部分后的音频信号在原始音频中的起始和结束索引。
## 帧长度和跳数
假设你有一个音频信号，采样率为 22050 Hz，帧长度为 2048 个样本，跳数为 512 个样本。这意味着：

- 每个帧包含 2048 个样本。
- 每次移动 512 个样本来获取下一个帧。

因此，第一个帧从样本 0 开始，到样本 2047 结束。第二个帧从样本 512 开始，到样本 2559 结束。第三个帧从样本 1024 开始，到样本 3071 结束，依此类推。
# 静音消除（包括音频内部的）
```python
librosa.effects.split(y, *, top_db=60, ref=<function amax>, frame_length=2048, hop_length=512,  
aggregate=<function amax>)
```
## 参数
同`trim`
## 输出
- **intervals**：二维 NumPy 数组，形状为 `(n, 2)`，表示分割后的音频片段的起始和结束索引。每一行 `[start, end]` 表示一个音频片段的起始和结束样本索引
# 音频合并
```python
librosa.effects.remix(y,intervals,*,align_zeros=True)
```
## 参数
- **y**：一维 NumPy 数组，表示音频信号的波形数据。
- **intervals**：二维 NumPy 数组，形状为 `(n, 2)`，表示要重新混合的音频片段的起始和结束索引。每一行 `[start, end]` 表示一个音频片段的起始和结束样本索引。
- **align_zeros**：布尔值，默认值为 `True`。如果为 `True`，则在拼接音频片段时，会尝试对齐零交叉点，以减少拼接处的不连续性。
## 输出
- **y_remixed**：一维 NumPy 数组，表示重新混合后的音频信号。
