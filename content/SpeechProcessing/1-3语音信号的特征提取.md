# 信号的频域分析(傅里叶)
```python
librosa.stft(y, n_fft=2048, hop_length=None, win_length=None, window='hann', center=True, pad_mode='reflect', normalized=False, onesided=True, dtype=None, realval=False, length=None, use_length=False, progress=False)
```
## 参数
- **y**：一维 NumPy 数组，表示音频信号的波形数据。
- **n_fft**：整数，表示每个 STFT 窗口使用的 FFT 点数。默认为 2048。通常设置为 2 的幂。
- **hop_length**：整数，表示窗口之间的步长（即重叠的样本数）。如果为 `None`，则默认为 `n_fft // 4`。
- **win_length**：整数，表示每个窗口中的样本数。如果为 `None`，则默认为 `n_fft`。
- **window**：字符串、NumPy 数组或 callable，表示窗口函数。默认为 `'hann'`（汉宁窗）。
- **center**：布尔值，如果为 `True`，则窗口以样本的中间为中心。默认为 `True`。
- **pad_mode**：字符串，表示填充模式。默认为 `'reflect'`，其他选项包括 `'constant'` 等。
- **normalized**：布尔值，如果为 `True`，则返回归一化的 STFT。默认为 `False`。
- **onesided**：布尔值，如果为 `True`，则返回单边频谱。默认为 `True`。
- **dtype**：数据类型，指定输出的数据类型。默认为 `None`（即 `y` 的数据类型）。
- **realval**：布尔值，如果为 `True`，则返回实数 STFT。默认为 `False`。
- **length**：整数，表示输出 STFT 的长度。如果为 `None`，则返回完整的 STFT。
- **use_length**：布尔值，如果为 `True`，则使用 `length` 参数来截断输出。默认为 `False`。
- **progress**：布尔值，如果为 `True`，则显示进度条。默认为 `False`。
## 输出
- **D**：二维复数 NumPy 数组，表示音频信号的 STFT。形状为 `(n, t)`，其中 `n` 是频率分量的数量，`t` 是时间帧的数量。 

# 幅度转dB

```python
librosa.amplitude_to_db(S, *, ref=1.0, amin=1e-5, top_db=80.0)
```

## 参数
- **S**：非负实数的 NumPy 数组，表示幅度谱（如 STFT 的幅度、梅尔频谱等）。
- **ref**：浮点数或可调用对象（如 `np.max`），用于 dB 计算的参考值。默认为 `1.0`。若为函数，则 `ref = ref(S)`。
- **amin**：浮点数，幅度的最小阈值，防止对零取对数。默认为 `1e-5`。
- **top_db**：浮点数或 `None`，动态范围上限（单位：dB）。结果将被裁剪为不低于 `max(S_db) - top_db`。默认为 `80.0`；若为 `None`，则不裁剪。

### 输出
- **S_db**：与输入 `S` 形状相同的 NumPy 数组，单位为分贝（dB），计算公式为：
$$
  S_{\text{db}} = 20 \cdot \log_{10}\left( \frac{\max(S, \text{amin})}{\text{ref}} \right)
$$
  若 `top_db` 非 `None`，还会进行动态范围裁剪。

# `librosa` 内置绘图

```python
librosa.display.specshow(
    data,
    *,
    x_axis=None,
    y_axis=None,
    sr=22050,
    hop_length=512,
    n_fft=None,
    win_length=None,
    fmin=0.0,
    fmax=None,
    tuning=0.0,
    bins_per_octave=12,
    key='C:maj',
    htk=False,
    cmap='magma',
    rasterized=True,
    ax=None,
    **kwargs
)
```

## 参数
- **data**：二维 NumPy 数组，表示频谱数据（如幅度谱、梅尔谱、MFCC 等），形状通常为 `(n_freq, n_time)`。
- **x_axis**：字符串或 `None`，指定 X 轴类型。常用值包括 `'time'`（秒）、`'frames'`、`'ms'` 等。
- **y_axis**：字符串或 `None`，指定 Y 轴类型。常用值包括 `'linear'`、`'log'`、`'mel'`、`'hz'`、`'chroma'` 等。
- **sr**：整数，音频采样率（Hz），用于自动计算时间与频率刻度。默认为 `22050`。
- **hop_length**：整数，STFT 帧移（样本数），用于时间轴计算。默认为 `512`。
- **n_fft**：整数，FFT 点数，用于频率轴刻度（当 `y_axis` 为 `'fft'` 或 `'linear'` 时）。默认为 `None`（自动推断）。
- **win_length**：整数，窗口长度（样本数），通常与 `n_fft` 一致。
- **fmin**：浮点数，显示的最小频率（Hz），对 `'mel'`、`'log'` 等 y 轴有效。默认为 `0.0`。
- **fmax**：浮点数或 `None`，显示的最大频率（Hz）。默认为 `None`（由 `sr` 决定）。
- **tuning**：浮点数，调音偏移（以音分为单位），用于音高对齐。默认为 `0.0`。
- **bins_per_octave**：整数，每八度音程的频带数，用于常 Q 变换相关显示。默认为 `12`。
- **key**：字符串，参考调性（如 `'C:maj'`），用于 chroma 显示。默认为 `'C:maj'`。
- **htk**：布尔值，是否使用 HTK 风格的梅尔滤波器。默认为 `False`。
- **cmap**：字符串，颜色映射方案（如 `'viridis'`, `'plasma'`, `'gray'`）。默认为 `'magma'`。
- **rasterized**：布尔值，是否将图像光栅化以减小文件大小（适用于矢量图输出）。默认为 `True`。
- **ax**：`matplotlib.axes.Axes` 对象，指定绘图坐标轴。若为 `None`，则使用当前轴。
- **\*\*kwargs**：其他传递给 `matplotlib.pyplot.pcolormesh` 的关键字参数（如 `vmin`, `vmax`）。

## 输出
- 返回 `matplotlib.collections.QuadMesh` 对象（可用于添加 colorbar）。
- **副作用**：在指定或当前 Matplotlib 坐标轴上绘制频谱图，并自动设置时间/频率轴标签与刻度。

# 预加重
弥补了高频部分的损耗，保护了声道信息 
```python
librosa.effects.preemphasis(y, *, coef=0.97, zi=None, return_zi=False)
```

## 参数
- **y**：一维 NumPy 数组，表示输入的音频波形信号。
- **coef**：浮点数，预加重滤波器的系数，控制高频提升强度。通常取值在 `[0.9, 1.0)` 之间，默认为 `0.97`。
- **zi**：NumPy 数组或 `None`，滤波器的初始状态（用于分段连续处理）。默认为 `None`，表示零初始状态。
- **return_zi**：布尔值，若为 `True`，则同时返回滤波器的最终状态。默认为 `False`。

## 输出
- 若 `return_zi=False`：  
  - **y_out**：一维 NumPy 数组，表示预加重后的音频信号，长度与输入 `y` 相同。
- 若 `return_zi=True`：  
  - **(y_out, zf)**：元组，包含预加重后的信号 `y_out` 和滤波器的最终状态 `zf`（可用于后续分段处理的初始状态）。

# 梅尔滤波器组

```python
librosa.filters.mel(sr, n_fft, n_mels=128, fmin=0.0, fmax=None, htk=False, norm='slaney', dtype=<class 'numpy.float32'>)
```

## 参数
- **sr**：整数，音频采样率（Hz），用于确定频率范围。通常为 `22050` 或 `16000`。
- **n_fft**：整数，FFT 的点数，用于确定频域分辨率（即线性频谱的 bin 数量为 `n_fft // 2 + 1`）。
- **n_mels**：整数，要生成的梅尔滤波器数量（即输出梅尔频带数）。默认为 `128`。
- **fmin**：浮点数，最低频率（Hz），低于此频率的成分将被忽略。默认为 `0.0`。
- **fmax**：浮点数或 `None`，最高频率（Hz）。若为 `None`，则默认为 `sr / 2`（奈奎斯特频率）。
- **htk**：布尔值，是否使用 HTK（Hidden Markov Model Toolkit）的梅尔频率公式。默认为 `False`（使用 Slaney 的听觉临界带公式）。
- **norm**：字符串或 `None`，滤波器归一化方式：
  - `'slaney'`（默认）：每个滤波器的面积归一化为 1（Slaney 风格），适用于 MFCC。
  - `None`：不归一化。
  - 其他值（如 `'ortho'`）目前不支持，仅 `'slaney'` 或 `None` 有效。
- **dtype**：数据类型，指定输出数组的数据类型。默认为 `np.float32`。

## 输出
- **M**：二维 NumPy 数组，形状为 `(n_mels, n_fft // 2 + 1)`，表示梅尔滤波器组矩阵。  
  每一行对应一个三角形梅尔滤波器，作用于线性频谱（如 STFT 的幅度谱或功率谱）以得到梅尔频谱：
$$
  \text{mel\_spectrogram} = M \cdot \text{linear\_spectrogram}
$$

# `librosa.feature.melspectrogram`（用于提取 Fbank 特征）

> **说明**：在语音处理中，**Fbank**（Filter Bank）特征通常指**梅尔滤波器组能量**（Mel-filterbank energies），即对音频信号进行 STFT → 功率谱 → 梅尔滤波器组加权 → 取对数（可选）得到的特征。LibROSA 中通过 `librosa.feature.melspectrogram` 实现核心步骤，再结合 `power_to_db` 可得对数 Fbank。

```python
librosa.feature.melspectrogram(
    y=None,
    sr=22050,
    S=None,
    n_fft=2048,
    hop_length=512,
    win_length=None,
    window='hann',
    center=True,
    pad_mode='reflect',
    power=2.0,
    n_mels=128,
    fmin=0.0,
    fmax=None,
    htk=False,
    norm='slaney',
    dtype=<class 'numpy.float32'>
)
```

## 参数
- **y**：一维 NumPy 数组，输入音频波形。若提供 `S`，则可为 `None`。
- **sr**：整数，音频采样率（Hz）。默认为 `22050`。
- **S**：二维 NumPy 数组，预计算的频谱（如幅度谱或功率谱）。若提供，则跳过 STFT 计算。形状应为 `(n_fft // 2 + 1, n_frames)`。
- **n_fft**：整数，FFT 点数。默认为 `2048`。
- **hop_length**：整数，帧移（样本数）。默认为 `512`。
- **win_length**：整数，窗长（样本数）。若为 `None`，则等于 `n_fft`。
- **window**：字符串或数组，窗函数类型。默认为 `'hann'`。
- **center**：布尔值，是否在信号两端填充使帧居中。默认为 `True`。
- **pad_mode**：字符串，填充模式（当 `center=True` 时使用）。默认为 `'reflect'`。
- **power**：浮点数，计算频谱时的幂次。  
  - `power=1.0`：使用幅度谱（|STFT|）  
  - `power=2.0`：使用功率谱（|STFT|²）← **Fbank 通常使用此项**
- **n_mels**：整数，梅尔滤波器数量（即 Fbank 维度）。常用值：`40`, `64`, `80`。默认为 `128`。
- **fmin**：浮点数，最低频率（Hz）。默认为 `0.0`。
- **fmax**：浮点数或 `None`，最高频率（Hz）。默认为 `sr / 2`。
- **htk**：布尔值，是否使用 HTK 梅尔公式。默认为 `False`。
- **norm**：字符串或 `None`，梅尔滤波器归一化方式。  
  - `'slaney'`（默认）：面积归一化（Slaney 风格）  
  - `None`：不归一化
- **dtype**：数据类型。默认为 `np.float32`。

## 输出
- **M**：二维 NumPy 数组，形状为 `(n_mels, n_frames)`，表示**梅尔频谱**（即 Fbank 能量）。  
  每一列对应一个时间帧，每一行对应一个梅尔频带的能量（线性尺度）。

# MFCC特征

```python
librosa.feature.mfcc(
    y=None,
    sr=22050,
    S=None,
    n_mfcc=20,
    dct_type=2,
    norm='ortho',
    lifter=0,
    **kwargs
)
```

## 参数
- **y**：一维 NumPy 数组，输入音频波形。若已提供频谱 `S`，可为 `None`。
- **sr**：整数，音频采样率（Hz）。默认为 `22050`。
- **S**：二维 NumPy 数组，预计算的**对数梅尔频谱**（Log Mel-spectrogram），形状为 `(n_mels, n_frames)`。若提供，则跳过梅尔谱计算。
- **n_mfcc**：整数，要保留的 MFCC 系数数量（即输出维度）。常用值为 `13` 或 `20`。默认为 `20`。
- **dct_type**：整数，DCT（离散余弦变换）类型。默认为 `2`（语音处理标准）。
- **norm**：字符串或 `None`，DCT 的归一化方式：
  - `'ortho'`（默认）：正交归一化（推荐，能量守恒）
  - `None`：无归一化
- **lifter**：整数，倒谱提升系数（用于平滑高频 MFCC）。若为 `0`（默认），则不应用提升。
- **\*\*kwargs**：其他传递给 `librosa.feature.melspectrogram` 的参数（如 `n_fft`, `hop_length`, `n_mels`, `fmin`, `fmax` 等），仅在 `S=None` 时使用。

## 输出
- **M**：二维 NumPy 数组，形状为 `(n_mfcc, n_frames)`，表示 MFCC 特征。  
  每一列对应一个时间帧，每一行对应一个 MFCC 系数（第 0 阶通常表示整体能量）。

## 📌 说明
**MFCC**（Mel-Frequency Cepstral Coefficients，梅尔频率倒谱系数）是语音和音频识别中最经典的特征之一，提取流程如下：

1. **预加重**（可选）：`y = librosa.effects.preemphasis(y)`
2. **分帧加窗 + STFT** → 得到频谱
3. **计算功率谱**：\|STFT\|²
4. **通过梅尔滤波器组** → 得到梅尔频谱（Fbank）
5. **取对数**：`log(Mel-spectrogram)` → Log Fbank
6. **对每个时间帧做 DCT** → 得到 MFCC

> `librosa.feature.mfcc` 内部自动完成步骤 2–6（若未提供 `S`）；若提供 `S`，则假设 `S` 已是对数梅尔谱，直接执行步骤 6。


## 特征差分

```python
librosa.feature.delta(
    data,
    *,
    width=9,
    order=1,
    axis=-1,
    mode='interp',
    **kwargs
)
```

### 参数
- **data**：二维 NumPy 数组，输入特征序列，形状通常为 `(n_features, n_frames)`。  
  例如：MFCC、梅尔频谱、Fbank 等随时间变化的特征。
- **width**：奇数整数，差分计算的窗口宽度（必须为奇数）。默认为 `9`。  
  控制平滑程度：值越大，差分越平滑；值越小，对局部变化越敏感。
- **order**：整数，差分阶数。  
  - `order=1`：一阶差分（速度，即特征变化率）  
  - `order=2`：二阶差分（加速度，即变化率的变化率）  
  默认为 `1`。
- **axis**：整数，沿哪个轴计算差分。默认为 `-1`（即时间轴，通常为最后一维）。
- **mode**：字符串，边界填充模式。默认为 `'interp'`（线性插值）。  
  其他选项：`'constant'`, `'nearest'`, `'reflect'`, `'wrap'`（同 `np.pad`）。
- **\*\*kwargs**：传递给 `scipy.ndimage.convolve1d` 的额外参数（如 `output`）。

### 输出
- **delta_feat**：与输入 `data` 形状相同的 NumPy 数组，表示差分特征。  
  - 若 `order=1`：返回一阶差分（动态特征，描述特征如何随时间变化）  
  - 若 `order=2`：返回二阶差分（加速度特征）

---

### 📌 作用与意义

在语音识别、说话人识别等任务中，**静态特征**（如 MFCC）只描述“当前帧是什么”，而**动态特征**（delta）描述“特征如何变化”，能显著提升系统性能。

| 特征类型 | 含义 | 常用组合 |
|--------|------|--------|
| **Static** | 原始特征（如 MFCC） | MFCC |
| **Delta**（Δ） | 一阶差分（变化速度） | MFCC + Δ |
| **Delta-Delta**（ΔΔ） | 二阶差分（变化加速度） | MFCC + Δ + ΔΔ |
