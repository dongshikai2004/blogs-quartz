# 📘 第2章 物理层 — 系统笔记

## 一、物理层的本质任务
> **核心问题**：  
> 计算机产生的数据如何在**介质上（机械）**、以**什么方式（电气）**、**有序地（时序）** 传输到另一台计算机？

### 四大接口特性
| 特性       | 说明                       |
| -------- | ------------------------ |
| **机械特性** | 接口形状/尺寸、引线数目与排列、固定/锁定装置等 |
| **电气特性** | 各线上电压范围                  |
| **功能特性** | 某电平在某线上表示何种意义            |
| **过程特性** | 不同功能事件的出现顺序（时序）          |

## 二、2.1 数据通信基础

### 2.1.1 基本概念

- **数据**：信息的载体  
  - *模拟数据*：连续（语音、温度）  
  - *数字数据*：离散（文字、整数）
- **信息**：数据的内涵
- **信号**：数据的电磁表现  
  - *模拟信号*：连续变化  
  - *数字信号*：离散取值

#### 通信系统模型
```
信源 → 输入信息 → 输入数据 → 发送器 → 发送信号 → 传输系统 → 接收信号 → 接收器 → 输出数据 → 输出信息 → 信宿
```
> 示例：PC → 调制解调器 → 公用电话网（模拟）→ 调制解调器 → PC

#### 基带 vs 宽带信号
| 类型       | 定义            | 特点                |
| -------- | ------------- | ----------------- |
| **基带信号** | 用两种电压直接表示 0/1 | 占用全频带，不调制，适合短距局域网 |
| **宽带信号** | 基带信号经调制后频分复用  | 可多路复用，提高线路利用率     |

### 2.1.2 奈奎斯特定理 & 香农定理

#### 📏 性能指标

| 指标        | 公式/说明                         |
| --------- | ----------------------------- |
| **传输时延**  | = 数据块长度 / 信息传输速率（发送时间）        |
| **传播时延**  | = 信道长度 / 电磁波传播速率（≈2×10⁸ m/s）  |
| **总时延**   | = 传输时延 + 传播时延 + 处理时延 + 排队时延   |
| **时延带宽积** | = 传播时延 × 带宽（单位：bit）→ 衡量链路"容量" |
| **RTT**   | 往返时延：发送 → 接收 → 确认返回           |

#### 📡 信号劣化三大因素
- **时延（delay）**
- **衰减（attenuation）**
- **噪声（noise）**

#### 🔊 带宽（Bandwidth）
振幅不会明显减弱的 **频率范围** 称为带宽
- 频域：**Hz**（频率范围）  
- 时域等效：**bps**（数据率能力）

#### 📊 速率概念辨析
码元：一个离散信号（电压）状态或信号事件
码元传输的速率越高，或信号传输的距离越远，在信道的输出端的波形的失真就越严重

| 术语              | 英文                      | 单位       | 定义                           |
| --------------- | ----------------------- | -------- | ---------------------------- |
| **信息传输速率**  $S$ | Bit Rate                | bps      | 每秒传输的**比特数**                 |
| **码元传输速率**  $B$ | Baud Rate               | Baud（波特） | 每秒传输的**码元数**（离散信号状态数）        |
| **关系**          | $S = B \times \log_2 V$ | —        | $V$：码元可能取值个数（如 4 相位 = $V=4$） |

> ✅ 示例：3 bit/码元 → 18 bit 数据 → 6 码元 → 同样波特率下速率提高 3 倍。

#### 🧠 奈奎斯特准则（无噪声理想信道）
> 理想低通信道最高码元速率：
$$
B_{\max} = 2H \quad \text{（Baud）}
$$
- $H$：信道带宽（Hz）  
- **注意**：实际信道远低于此值（因失真/干扰）

#### 📈 香农公式（有噪声实际信道）
> 信道极限信息传输速率（无差错）：
$$
C = W \log_2\left(1 + \frac{S}{N}\right) \quad \text{（bps）}
$$
- $W$：带宽（Hz）  
- $S/N$：信噪比（功率比）
	- $S$ ：信道内所传信号的平均功率
	- $N$ ：信道内部的高斯噪声功率
- 信噪比（dB）：$10 \log_{10}(S/N)$  
  - $S/N=10$ → 10 dB；$S/N=1000$ → 30 dB

> ✅ 启示：  
> - 提升速率途径：增带宽、提信噪比、增每码元比特数  
> - 实际速率 << $C$

#### 🔁 两定理作用位置
- **奈氏**：限制**码元速率**（发送器→信道）  
- **香农**：限制**信息速率**（信道本质容量）

### 2.1.3 数据传输方式

| 类型      | 说明                   | 图示         |
| ------- | -------------------- | ---------- |
| **单工**  | 单向传输（如广播）            | A → B      |
| **半双工** | 可收可发，但**不能同时**（如对讲机） | A ⇄ B（交替）  |
| **全双工** | **同时**收发（如电话）        | A ↔ B（双通道） |

| 通信方式 | 原理 | 特点 |
|----------|------|------|
| **串行通信** | 比特**逐位**传输（1条线） | 距离远、成本低、速率可高（如USB/RS232） |
| **并行通信** | 多位**同时**传输（n条线） | 距离短、易干扰（时钟偏移）、成本高 |

### 2.1.4 调制解调技术（数字→模拟）

#### 📡 调制解调器（Modem）
- **调制器（MOdulator）**：数字 → 模拟（0/1 → 300~3400 Hz 音频信号）  
- **解调器（DEModulator）**：模拟 → 数字  
> 作用：波形变换器（调制） + 波形识别器（解调）

#### ⚙️ 基本调制方法（二元制）
| 方法 | 缩写 | 原理 |
|------|------|------|
| 调幅 | ASK | 振幅随数据变（0/1 对应不同振幅） |
| 调频 | FSK | 频率随数据变 |
| 调相 | PSK | 相位随数据变 |

#### 🌐 高级调制：QAM（正交振幅调制）
- 同时调**幅+调相**，形成星座图  
- 例：16-QAM → 16种状态 → **4 bit / 码元**  
- ⚠️ 码元状态越多 → 抗干扰越差 → 误码率↑

### 2.1.5 编码解码技术（基带传输）

> 基带传输 = 数字数据 → 数字信号 → 直接传输（无需载波）

#### 常用编码方式

| 编码 | 原理 | 优缺点 | 应用 |
|------|------|--------|------|
| **NRZ（不归零）** | 高电平=1，低电平=0 | ❌ 无自同步、有直流分量 | 早期设备 |
| **曼彻斯特码** | 位中跳变：<br>↑=0，↓=1 | ✅ 自同步、无直流<br>❌ 效率50%（信号频率=数据率×2） | 以太网（10BASE-T） |
| **差分曼彻斯特** | 位中必跳变（时钟）；<br>位首跳变=0，无跳变=1 | ✅ 时钟/数据分离更清晰 | 令牌环网 |

> 🎯 关键特性：  
> - 含时钟信息 → 实现**自同步**  
> - 无直流分量 → 减少损耗  
> - "违例码"（V0/V1）可用于帧起止标志
