# 网络中的网络NiN  
LeNet、AlexNet和VGG都有一个共同的设计模式：通过一系列的卷积层与汇聚层来提取空间结构特征；然后 通过全连接层对特征的表征进行处理  

**NiN** 在每个像素的通道上分别使用多层感知机  
## NiN块

使用 $1\times 1$ 卷积层充当带有ReLU激活函数的逐像素全连接层

# 批量规范化  
加速深层网络的收敛速度  
## 训练深层网络  
挑战  
1. 数据预处理的方式会对结果产生巨大影响  
2. 中间层的变量可能有更广的变化范围  
3. 容易过拟合  

批量规范化应用于单个可选层（也可以所有层），原理：  
1. 规范化输出，减去均值并除以方差，基于当前小批量的处理  
2. 应用比例系数和比例偏移  

$\mathbf{x}\in \mathcal B$ 表示来自小批量$\mathcal B$ 的输入，批量规范化BN按以下表达式转换$\mathbf x$  
$$  
\mathrm{BN}({\bf x})=\gamma\,{\bf\odot}\,\frac{{\bf x}-\hat{\mu}_{\mathcal B}}{\hat{\sigma}_{B}}+\beta  
$$  

参数（形状与 $\mathbf x$ 相同）  
- $\gamma$ 拉伸参数  
- $\beta$ 偏移参数  
其中  
- $\hat \mu_{\mathcal B}$ 小批量样本$\mathcal B$ 的均值  
- $\hat \sigma_{\mathcal B}$ 小批量样本$\mathcal B$ 的样本标准差  
$$  
\begin{array}{l}  
{{\hat{\mu}_{B}={\frac{1}{|{\cal B}|}}\sum_{\mathbf{x}\in \mathcal B}\mathbf{x},}}\\ {{\hat{\sigma}_{B}^{2}={\frac{1}{|{\cal B}|}}\sum_{\mathbf{x\in\mathcal B}}(\mathbf{x}-{\hat{\mu}}_{\mathcal B})^{2}+\epsilon}}  
\end{array}  
$$  
方差均值添加常量 $\epsilon >0$ ，以确保不会除以0  
噪声是有益的  
## 批量规范化层  
全连接层和卷积层的批量规范化略有不同  
### 全连接层  
将 批量规范化 应用在 仿射变换 和激活函数之间  
$$  
{\bf h}=\phi(\mathrm{BN}({\bf W x}+{\bf b}))  
$$  
### 卷积层  
在卷积层之后和非线性激活函数之前应用批量规范化  
卷积有多个输出通道时，需要对每个通道输出作批量规范化，每个通道有自己的拉伸和偏移参数  
假设卷积的输出 高度 $p$ 宽度 $q$ ，小批量包含 $m$个样本，对于卷积层，每个输出通道的 $m\times p\times q$ 个元素上同时作批量规范化  
### 预测中的批量规范化  
批量规范化在训练和预测模式下的行为不同  
预测时  
- 不需要样本均值的噪声和样本方差  
- 逐个预测时，  
	- 移动整个训练样本的样本均值和方差  
# 残差网络ResNet  
## 函数类  
神经网络框架 $\mathcal F$ ,对于所有的 $f\in \mathcal F$,存在一些参数集可以通过训练获得  
假设 $f^*$ 是想获得的函数，如果 $f^*\in \mathcal F$，那么可以通过训练获得  
如果找不到，可以尝试找到 $f^*_{\mathcal F}$ 是 $\mathcal F$ 中的最佳选择，可以通过优化问题来解决  
$$  
f^*_{\mathcal{F}}:=\arg\operatorname*{min}_{f}L(\mathbf{X},\mathbf{y},f)\;\mathrm{subject\;to}\;f\in\mathcal{F}  
$$  
需要设计更强大的架构 $\mathcal F^\prime$ 使 $f^*_{\mathcal F}$比 $f^*_{\mathcal F^\prime}$ 更近似  
![[Pasted image 20250822214546.png]]  
左图越来越偏移  
只有嵌套类的可以避免 $f^*_{\mathcal F}$ 更糟的问题  
对于深度神经网络，如果将新添加的层训练为恒等映射 $f(\mathbf x)=\mathbf x$ ，新模型将和原有模型一样有效  
新模型可以得出更优解来拟合，新添加的层更容易降低训练误差  
残差网络的**核心思想**是：每个附加层都应该更容易地包含原始函数作为其元素之一

# 稠密连接网络DenseNet  
## 从ResNet到DenseNet  
泰勒展开  
$$  
f(x)=f(0)+f^{\prime}(0)x+\frac{f^{\prime\prime}(0)}{2!}x^{2}+\frac{f^{\prime\prime}(0)}{3!}x^{3}+.\cdot\cdot\cdot  
$$  
ResNet将函数展开为  
$$  
f({\bf x})={\bf x}+g({\bf x})  
$$  
一个简单的线性项和复杂的非线性项![[Pasted image 20250822215511.png]]  
左ResNet,右DenseNet  
$$  
\mathbf{x}\rightarrow[\mathbf{x},f_{1}(\mathbf{x}),f_{2}([\mathbf{x},f_{1}(\mathbf{x})]),f_{3}([\mathbf{x},f_{1}(\mathbf{x}),f_{2}([\mathbf{x},f_{1}(\mathbf{x})])]),\cdot\cdot\cdot\cdot]  
$$  
将这些展开式结合到多层感知机中，再次减少特征的数量  
![[Pasted image 20250822220458.png]]  
## 稠密块体  
基础卷积块  
1. 批量规范化  
2. 激活  
3. 卷积  

``` python  
def conv_block(num_channels):   
	blk = nn.Sequential()   
	blk.add(nn.BatchNorm(),   
			nn.Activation('relu'),   
			nn.Conv2D(num_channels, kernel_size=3, padding=1)  
	)   
	return blk  
	
class DenseBlock(nn.Block):   
	def __init__(self, num_convs, num_channels, **kwargs):   
	super().__init__(**kwargs)   
	self.net = nn.Sequential()   
	for _ in range(num_convs):   
		self.net.add(conv_block(num_channels))   
	def forward(self, X):   
		for blk in self.net:   
			Y = blk(X)   
			X = np.concatenate((X, Y), axis=1)  
		return X  
```  
## 过渡层  
控制模型复杂度  
用 $1\times 1$ 卷积层减小通道数  
步幅为 $2$ 的平均汇聚层减半高和宽  
```python  
def transition_block(num_channels):   
	blk = nn.Sequential()   
	blk.add(nn.BatchNorm(),   
			nn.Activation('relu'),   
			nn.Conv2D(num_channels, kernel_size=1),   
			nn.AvgPool2D(pool_size=2, strides=2))   
	return blk  
```
