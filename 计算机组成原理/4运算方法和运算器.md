# 基本算术运算的实现
最基本的运算部件--加法器
## 基本运算部件
加法器 由 全加器 和其他必要逻辑电路组成
全加器(FA)
- 三个输入量
	- 操作数 $A_i$ 和 $B_i$ ,低位传入的进位 $C_i$
- 两个输出量
	- 本位和 $S_i$
	- 向高位的进位 $C_i$
![[Pasted image 20251214165343.png]]
### 串行加法器 
只有一个全加器
数据逐位串行送入加法器进行运算
### 并行加法器
多个全加器组成
位数的多少取决于机器的字长，数据的各位同时运算
最长运算时间主要是由 进位信号的传递时间决定
## 进位的产生与传递
$C_i=A_i B_i+(A_i⊕B_i)C_{i-1}$
将本位和进位分开
$C_i=G_i+P_iC_{i-1}$

# 定点数的移位运算与舍入操作
## 无符号数的移位
空出位都以 $0$ 补,移出位丢失
## 带符号数的移位
1. 正数与 无符号数 相同
2. 负数 左移末尾补0,右移开投补1
## 移位功能的实现
![[Pasted image 20251215194617.png]]
移位器
与移位寄存器不同,本身只有移位功能,没有寄存功能
移位之后的结果必须保存到寄存器中
## 舍入操作
1. 恒舍(切断)
	1. 误差大
2. 冯诺依曼舍入法(恒置 $1$)
	1. 最后 $1$ 位始终为1
	2. 误差小
	3. 简单
	4. 应用多
3. 下舍上入法
	1. 舍去的最高位作为标志
	2. 降低误差,需增加硬件
4. 查表舍入法(ROM舍入法)
	1. 用ROM存放舍入处理表,每次查表来处理
	2. 速度快
	3. 增加硬件
	4. 整体表现最佳

# 运算器的基本组成
运算器不仅可以完成数据信息的算术逻辑运算，还可以作为 **数据信息的传送通路**
## 运算器结构
包含 实现基本算术和逻辑运算功能的ALU、提供操作数与暂存结果的寄存器组、有关的判别逻辑和控制电路
构成整体时需解决问题:如何向ALU提供操作数
解决方法:在ALU输入端加多路选择器/在ALU输入端加一级锁存器(暂存器)
运算器内的各功能模块之间的连接也广泛采用总线结构，这个总线称为**运算器的内部总线** ,与系统总线的含义不同，运算器的内部总线是CPU的内部数据通路，因此只有数据线
### 带多路选择器的运算器
![[Pasted image 20251215214131.png]]
各寄存器独立,多路地将数据送至ALU的多路选择器,ALU有选择地接收两路输入数据.
运算器的内部总线 一组 单向传送 的数据线,将结果送至寄存器,
由寄存器的同步打入脉冲将内部总线上的数据送入$R_i$。如果 同时发出几个打入脉冲，则可将总线上的同一数据同时送入几个相关的寄存器中
### 带输入锁存器的运算器
![[Pasted image 20251215214526.png]]
运算器的内部总线是双向的

运算器的内部总线结构
1. 单总线结构运算器
	1. 上图
	2. 运算器实现一次双操作数的运算需要分成 $3$ 步
	3. 慢
2. 双总线结构运算器
	1. 两个操作数分别通过总线12同时送至ALU中计算,立即得到运算结果
	2. ALU的输出不能直接送到总线,两条总线都被操作数占据,必须在ALU输出端设置缓冲器,再将结果送至寄存器
	3. 比单总线少一步,2步
3. 三总线结构运算器
	1. 两个输入和一个输出各有一条总线
	2. 如果某一个数不需要运算或修改,可以直接通过总线旁路器把数据送出
	3. 可以在一步中完成
	4. 操作速度快
	5. 控制复杂
